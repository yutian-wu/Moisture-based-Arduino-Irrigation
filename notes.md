## Development Board

| Aspect      | Value |
| ----------- | ----------- |
| Microcontroller      | ATmega328       |
| Operating Voltage   | 5V        |
| Input Voltage (recommended)   | 7-12V        |
| Digital IO Pins   | 14 (6 PWM outputs)        |
| Analog Input Pins   | 6        |
| Flash Memory   | 32 KB        |
| Clock Speed   | 16 MHz        |
